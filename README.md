# Microbackends DSIII

Este proyecto contiene un backend en Node.js con TypeScript que expone un CRUD de palabras conectado a Supabase (Postgres).

## 🚀 Tecnologías
- Node.js + Express
- TypeScript
- Supabase (Postgres)
- Vercel (despliegue)

## 📦 Instalación
```bash
npm install
```

## 🌐 Endpoints
POST /words → Inserta una palabra

GET /words → Lista todas las palabras
